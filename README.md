# smart-meter-energy-fault-dataset
A dataset contain dataset contains energy consumption information collected from residential homes across London (UK Data Service 2014) and is a subset of the London Smart Meter Dataset which was taken between the year 2011 to 2014.
300 meters were selected are random from this huge london energy used over the range of one year. 
The dataset contains 4 major fault such as, noise, drop, stuck and spike.
The fault were applied randomly and account for 20% OF the overall dataset. This dataset can be used freely for other research works
